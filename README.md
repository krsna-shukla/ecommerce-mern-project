# ShopEasy - MERN E-Commerce Website

A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The platform supports user authentication, product browsing, shopping cart management, order placement, and admin product/order management.

## Live Demo

Frontend: https://ecommerce-mern-project-sand.vercel.app/

Backend API: https://ecommerce-mern-project-dimt.onrender.com

## Features

### User Features

* User Registration & Login
* JWT Authentication
* Browse Products
* Search Products
* Category Filtering
* Product Details Page
* Add to Cart
* Update Cart Quantity
* Remove Items from Cart
* Place Orders
* View Order History

### Admin Features

* Admin Dashboard
* Add Products
* Delete Products
* View All Orders
* Update Order Status
* Role-Based Access Control

## Tech Stack

### Frontend

* React.js

### Backend

* Node.js

## Installation

### Clone Repository

git clone https://github.com/krsna-shukla/ecommerce-mern-project.git

### Backend Setup

cd server
npm install
npm run dev

### Frontend Setup

cd frontend
npm install
npm run dev
